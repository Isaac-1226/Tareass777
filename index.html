HTML,CSS y JavaScript

Son los tres pilares fundamentales del desarrollo web que trabajan juntos para crear páginas web dinámicas e interactivas

HTML se encarga de la estructura y el contenido, CSS del diseño y la presentación, y JavaScript de la interactividad y la funcionalidad.

Definiciones:

HTML (HyperText Markup Language):

Es el lenguaje de marcado que define la estructura y contenido de una página web. Utiliza etiquetas para delimitar diferentes elementos, como párrafos, enlaces, tablas, imágenes, etc.

CSS (Cascading Style Sheets):

Es un lenguaje de estilos que se utiliza para controlar la presentación visual de una página web. Permite definir aspectos como los colores, fuentes, tamaños, márgenes, etc., de los elementos HTML.

JavaScript (Program. Language):

Es un lenguaje de programación que permite agregar funcionalidad interactiva a una página web. Permite manipular los elementos HTML, responder a eventos del usuario, realizar cálculos, validar formularios, entre otros

Analogia de la web y una casa:

HTML

Estructura de la casa.

Es como los cimientos, las paredes, techos, y distribucion de las habitaciones

HTML define la estructura basica de una pagina web: que elementos hay(titulo, parrafos, botones, imagenes, etc.), pero sin preocuparse por como se ven.

CSS

Decoracion y estilo de la casa.

Es la pintura de las paredes, el tipo de piso, los muebles, las cortinas, el diseño de la cocina

CSS se encarga del aspecto visual de la página: colores, márgenes, fuentes, tamaños, estilos responsivos… lo visual.

JavaScript

Los sistemas electricos y electronicos de la casa.

Es como los interruptores de luz, los enchufes, las puertas automáticas, el termostato inteligente

JavaScript añade interactividad y funcionalidad haciéndola más dinámica, con respuesta a un evento

Resumen

"Los tres lenguajes son esencuales para el desarrollo web, y cada una juega un papel crucial en la creacion de paginas web modernas y funcionales"
